DBAcademy Course, Build, Test & Other Tools

  • Python Docs for the DBAcademy library
  •  
  • EC 01 - Your First Lesson: Demonstrate how a typical lesson should be constructed.
  • EC 01L - Your First Lab: Demonstrates how a typical lab should be constructed including usage of #TODO and #ANSWER cells.
  • EC 02 - DBAcademyHelper: Introduces the DBAcademyHelper object and demos commonly available variables, path variables and injection of values into SQL statements.
  • EC 03 - Build Directives: Introduces build directives such as #SOURCE_ONLY, #TODO, #ANSWER, #DUMMY, #INCLUDE_HEADER_TRUE, #INCLUDE_HEADER_FALSE, #INCLUDE_FOOTER_TRUE, #INCLUDE_FOOTER_FALSE and the i18n directives used for translating courseware.
  • EC 04 - Do Not Publish Me - WIP: Introduces the work-in-progress concept to exclude notebooks from automatic inclusion by the build system.
  • EC 05 - Build Time Substitutions: Introduces build-time substitutions and in general how they are used.
  • EC 06 - Installing & Validating DatasetsIntroduces the patterns for installing and validating datasets along with utility methods for enumerating datasets to meet run-time requirements.
  • EC 07 - Working with Unity Catalog: Introduces functionality specifically around UC and configuring the environment with default catalogs.
  • EC 10.A - Testing DLT Notebooks: Demonstrate patterns for DLT Pipelines to facilitate automated testing.
  • EC 11 - Testing Multi-Task Jobs: Demonstrate patterns for Workflows (previously multi-task jobs) to facilitate automated testing.
  • EC 12 - Testing Multi-Task Jobs with Pipelines: Demonstrates how to combine both DLT and Workflows to facilitate automated testing.
  • EC 14 - Testing Streaming Notebooks: Demonstrates how to write streaming jobs to facilitate automated testing.
